---
description: Professional task management with todolist - break down work into small morsels, use stack-based pre-requirement handling, and scan for red flags after each task
---
1. Make professional, simple todolist to keep track of each task professionally.
2. As you go along todolist, break current task into smaller tasks as needed. Never take on too much at once. Work in small morsels, properly sequenced, one at a time. 
3. Treat each smaller task (from how you broke up the current todolist task) like a stack of small morsels - if need be, you can add new small morsel at the top of the stack, when you detect a pre-requirement. 
3.a Example: working on basketReservation schema structure, you might notice you need means to access the schema, which might lead you to add small pre-requirement morsel to be done first, which in that case would be to figure out access to schema and where schema is. Etc.
4. After each small task, scan for red flags: gaps, misunderstandings, omissions, mix ups, mistakes, mis-alignments, disconnections, false positives.
